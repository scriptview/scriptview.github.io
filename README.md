# ScriptView

A simple, elegant iPhone app for reading screenplays in the Fountain format.

## Features

- **Fountain format support** — Scene headings, dialogue, action, transitions, and more
- **Reading progress** — Automatically saves your position in each script
- **Easy import** — Via AirDrop, iCloud, or the Files app
- **Offline reading** — No internet connection or account required
- **Eye-friendly design** — Warm, paper-like aesthetic to reduce eye strain
- **Completely free** — No ads, no in-app purchases

## Privacy

ScriptView collects no user data. Your scripts and reading preferences stay on your device.
